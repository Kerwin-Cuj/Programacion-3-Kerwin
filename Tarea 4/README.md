Este programa implementa un Árbol AVL en Python que se balancea automáticamente mediante rotaciones. Utiliza herencia (clase AVL extiende de ABB) y permite la visualización gráfica de la estructura.

Funcionalidades:
-Inserción y Balanceo: Mantiene el árbol optimizado tras cada ingreso.
-Búsqueda: Localización rápida de valores.
-Carga Masiva: Lectura de datos desde archivos .csv.
-Graphviz: Generación y apertura automática de la imagen del árbol.

Estructura de Archivos
-Arbol_AVL_Kerwin.py: Menú principal y control.
-arboles.py: Lógica de nodos, ABB y rotaciones AVL.
-utils.py: Manejo de archivos CSV y exportación de imágenes.
-.csv: Archivos con datos de prueba.

Consta de un menú con varias opciones:
El usuario puede ingresar los numeros que desea y al seleccionar la opcion para generar imagen, se le mostrará en pantalla el resultado del árbol AVL, tambien puede seleccionar archivos .csv que estan creados (datos1.csv) y luego generar la imagen para ver el resultado del árbol AVL.

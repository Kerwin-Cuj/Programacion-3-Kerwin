Descripción:

Para este proyecto, usaé funciones de recursividad para resolver los problemas. 

1.-Primero, para el Binario, lo que hace el programa es ir dividiendo el número entre 2 una y otra vez. 
Los residuos que van quedando se van juntando para armar el número final. 

2.-En el Conteo de Dígitos es parecido: 
El programa le va quitando el último número a la cifra (dividiendo entre 10) y va sumando "1" cada vez que lo hace, 
hasta que ya no quedan más números que quitar.

3.-Para la Raíz Cuadrada, usé con una función mas, el programa empieza a probar números desde el cero en adelante (0, 1, 2, 3...) 
y se detiene justo antes de que el cuadrado del número se pase del valor ingresado.

4.-Luego, en la Conversión de Romanos, el código va revisando letra por letra; si ve que una letra pequeña está antes de una grande 
(como en el IV), sabe que tiene que restar, y si no, simplemente va sumando los valores de las letras.

5.-Por último, la Suma de Enteros es la más sencilla: el programa agarra el número que puse y le suma el anterior, 
y a ese el anterior, así hasta llegar a cero para dar el total de toda la suma acumulada. 

Todo esto corre dentro de un menú que se repite hasta que el usuario decide salir.
